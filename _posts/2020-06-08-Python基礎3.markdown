## 內積運算  
定義兩個向量$A(x1, y1, z1)$與$B(x2, y2, z2)$的內積運算為：
$ AB = (x1, y1, z1)  (x2, y2, z2) = x1x2 + y1y2 + z1z2 $


```python
%matplotlib inline
import numpy as np
import matplotlib.pyplot as plt
```
